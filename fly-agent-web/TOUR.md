```

│  [≡] Fly Agent          🎨              ⚙️              │  ← 顶部栏
├──────────┬──────────────────────────────────────────────┤
│          │  背景渐变 + 呼吸动画光晕                       │
│  侧边栏   │                                              │
│          │  ┌────────────────────────────────────┐      │
│ 📱 Logo  │  │                                    │      │
│          │  │        欢迎使用 Fly Agent          │      │
│ [+ 新建] │  │                                    │      │
│          │  │    向 AI 智能助手发送消息开始对话   │      │
│ ──────── │  └────────────────────────────────────┘      │
│ 📝 对话1 │                                              │
```

## 🎭 动画效果

### 1. 背景呼吸动画
- **位置**: 整个页面背景
- **效果**: 2个渐变圆形光晕,缓慢缩放和透明度变化
- **时长**: 8-10秒循环
- **颜色**: primary-200、primary-300

### 2. 消息气泡动画
- **入场**: 从下方出现,配合缩放效果
- **参数**: scale(0.95→1), y(10px→0)
- **时长**: 0.3秒
- **缓动**: ease-out

### 3. 侧边栏动画
- **展开/收起**: 弹簧动画
- **参数**: stiffness=300, damping=30
- **效果**: 平滑、自然的宽度变化

### 4. 按钮交互
- **悬停**: scale(1.02)
- **点击**: scale(0.98)
- **时长**: 200ms

### 5. 加载动画
- **效果**: 3个圆点上下跳动
- **延迟**: 每个点延迟0.1秒
- **时长**: 0.6秒循环

### 渐变应用
- **按钮**: from-primary-500 to-primary-600
- **背景**: from-primary-50 via-white to-primary-100
- **阴影**: shadow-primary-500/30

## 📱 响应式设计

### 桌面端 (≥768px)
- 侧边栏宽度: 280px
- 消息最大宽度: 80%
- 输入框最大宽度: 4rem

### 移动端 (<768px)
- 侧边栏默认折叠
- 消息最大宽度: 90%
- 全屏输入体验

## 🔍 交互细节

### 1. 输入框
- 自动高度 (最小48px, 最大200px)
- Enter 发送, Shift+Enter 换行
- 有内容时显示清空按钮
- 发送时禁用并显示加载图标

### 2. 会话管理
- 点击会话项切换
- 悬停显示删除按钮
- 删除前确认对话框
- 自动保存到 localStorage

### 3. 消息操作
- 自动滚动到底部
- 时间戳智能格式化
- 发送状态指示 (发送中/成功/失败)
- 流式输出打字机效果

### 4. 侧边栏
- 点击菜单按钮切换
- 弹簧动画平滑过渡
- 新建会话按钮在顶部
- 会话按时间倒序排列

## 💡 特殊效果

### 玻璃拟态 (Glassmorphism)
- 顶部栏: bg-white/80 backdrop-blur-lg
- 侧边栏: bg-white/80 backdrop-blur-lg
- 输入区: bg-white/80 backdrop-blur-lg
- 效果: 半透明 + 模糊背景

### 阴影层次
- 按钮阴影: shadow-primary-500/30
- 悬停阴影: shadow-primary-500/40
- 卡片阴影: shadow-sm
- 消息阴影: shadow-sm

### 圆角设计
- 按钮圆角: rounded-lg (8px)
- 输入框圆角: rounded-xl (12px)
- 消息圆角: rounded-2xl (16px)
- 特殊: 去掉消息靠近侧的圆角

## 🎯 用户体验

### 加载状态
- 发送消息时按钮显示加载图标
- AI 思考时显示跳动动画
- 流式输出时显示闪烁光标

### 空状态
- 无消息时显示欢迎界面
- 无会话时显示空列表提示
- 使用图标和文字引导用户

### 错误处理
- 发送失败显示错误图标
- 网络错误时友好提示
- 提供重试机制

### 反馈机制
- 按钮悬停视觉反馈
- 点击动画反馈
- 发送状态实时更新
- 时间戳相对显示
